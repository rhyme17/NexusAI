# NexusAI 用户使用手册

## 1. 适用对象

本手册面向普通用户和管理员，说明如何在前端完成登录、任务提交、任务执行与结果查看。

## 2. 登录与退出

1. 打开站点登录页（例如 `https://nexusai.rhyme17.top/login`）。
2. 输入用户名和密码登录。
3. 登录后进入系统首页（总览页）。
4. 退出登录：点击右上角用户菜单中的退出，系统会跳转到登录页。

## 3. 页面说明

- `/` 总览：查看服务健康状态、任务统计、最近任务。
- `/tasks` 任务台：创建任务、筛选任务、进入任务详情。
- `/tasks/[taskId]` 单任务工作区：执行任务、查看流程、查看结果和事件。
- `/agents` 智能体：查看智能体列表、状态和技能。
- `/settings` 设置：语言、API Key、执行偏好等配置。

## 4. 新建任务

1. 进入 `任务台`。
2. 点击创建任务。
3. 输入任务目标（例如“请输出一份关于 xx 的调研报告”）。
4. 选择优先级。
5. 提交后自动跳转到该任务详情页。

## 5. 执行任务

在任务详情页的执行区域：

1. 选择执行模式：
   - `single`：单智能体执行。
   - `pipeline`：多智能体按顺序执行。
   - `parallel`：多智能体并行执行。
2. 如需要，先点预览执行，确认步骤。
3. 点击开始执行。
4. 页面会实时显示事件流与状态变化。

## 6. 查看结果

任务执行完成后，在结果区域可看到：

- 结果正文
- 执行元数据（模型、耗时、token 等）
- 协作流程与事件记录

如果任务失败：

- 查看错误提示
- 根据提示调整后重试

## 7. API Key（用户侧）

用户侧 API Key 用于真实模型调用（非模拟结果）。

1. 打开 `设置` 页面。
2. 在用户 API Key 区域填入个人可用的模型访问令牌。
3. 保存后再次执行任务。

> 未配置可用 API Key 时，真实执行可能失败或回退到模拟路径。

## 8. 常见问题

### 8.1 任务一直执行中

- 刷新任务详情页。
- 检查事件流是否仍有新事件。
- 若长时间无变化，联系管理员检查后端日志。

### 8.2 结果为空

- 优先检查用户 API Key 是否有效。
- 检查模型服务是否可访问。
- 重试一次并查看错误信息。

### 8.3 登录失败

- 确认账号密码。
- 若忘记密码，请联系管理员重置。

## 9. 用户最佳实践

- 任务描述尽量具体：目标、范围、输出格式写清楚。
- 对报告类任务，建议在目标中明确“按 Markdown 输出完整报告”。
- 出现失败时先看错误类别（配置/模型服务/空结果），再决定重试或修改任务。
